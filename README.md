## 核心模块说明
###  左侧导航模块
- 功能：桌面端核心导航入口，固定展示4个一级菜单
- 包含项：门户（首页）、消息、通讯录、工作圈
- 交互逻辑：点击菜单切换主内容区，当前选中项高亮显示
- Tauri 特性适配：支持快捷键快速唤起（如 Alt+1 打开门户）
